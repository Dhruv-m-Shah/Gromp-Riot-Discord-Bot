Commands

All of GROMPS commands begin with a period ".", they are simple, intuitive, and easy to use. Heres the full list (More to come):

.match_history <Enter Summoner Name>

Returns the last 10 games a player has played. Includes KDA, Gold Earned, and Total Damage Dealt

.rank <Enter Summoner Name>

Returns ranked flex and ranked solo stats

.profile <Enter Summoner Name>

Returns bar graph and picture of top played champions

.rotation

Returns champions in the rotation

.random

Returns a random champion

.help

Returns a link to this site
